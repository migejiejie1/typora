```basic
win10 命令行 kms激活

1. 管理员身份运行 cmd

2. 卸载之前的密钥: slmgr /upk，弹出窗口显示“成功地卸载了产品密钥”;【如果提示找不到密钥，不用管，直接跳到下一步】

3. 安装产品密钥: slmgr /ipk NPPR9-FWDCX-D2C8J-H872K-2YT43，弹出窗口显示“成功地安装了产品密钥”

4. 设置密钥管理服务器: slmgr /skms zh.us.to，弹出窗口提示：“密钥管理服务计算机名称成功地设置为 zh.us.to”，网络上非常多这种服务器

5. 激活产品: slmgr /ato，弹出窗口提示：“成功地激活了产品”

6. 查看激活过期时间: slmgr /xpr

7. 查看激活信息: slmgr /dlv
```
