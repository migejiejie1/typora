# 密码中受支持的特殊字符

## Linux 系统上密码中受支持的特殊字符

下表提供在 Linux 系统上的密码中支持的特殊字符列表：

| 特殊字符的名称 | 特殊字符 |
| :------------- | :------- |
| 波浪线         | ~        |
| at 符号        | @        |
| 散列标记       | #        |
| 下划线         | _        |
| 插入标记       | ^        |
| 星号           | *        |
| 百分号         | %        |
| 斜杠           | /        |
| 句点           | .        |
| 加号           | +        |
| 冒号           | :        |
| 分号           | ;        |
| 等号           | =        |
